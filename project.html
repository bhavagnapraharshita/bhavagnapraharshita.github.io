Overview

Project Overview

Cloud cover is one of the biggest challenges in optical satellite remote sensing — it obscures ground features and significantly reduces the accuracy of land use classification. This M.Tech thesis research, conducted in collaboration with NRSC/ISRO, investigated and compared three different cloud removal techniques applied to Sentinel-2 and Landsat-8 satellite imagery.

The study evaluated how effectively each method could reconstruct cloud-affected pixels and improve the overall accuracy of supervised land cover classification across three major urban and peri-urban regions in Andhra Pradesh, India.

Objectives

Research Objectives

  • Identify the Area of Interest in satellite imagery and obtain ground truth points
  • Classify images and compare results against ground truth for accuracy assessment
  • Apply Image Fusion technique (Brovey) to remove cloud cover from Sentinel-2 imagery
  • Apply Conditional Algorithm (SQL threshold) for cloud pixel replacement
  • Apply ECDR (Efficient Cloud Detection and Removal) Algorithm for cloud reconstruction
  • Compare classification accuracy across all three methods and the original clouded image

Methodology

Three Cloud Removal Methods

01

Brovey Image Fusion

Fuses Sentinel-2 multispectral (10m) with Landsat-8 PAN (15m) to produce a cloud-free, high-resolution output.

  • Load Sentinel-2 (bands 2,3,4,8) — clouded image
  • Load Landsat-8 PAN — cloud-free image
  • Compress Landsat-8 PAN from 15m → 10m in ERDAS
  • Apply Brovey fusion to produce cloud-removed image
  • Apply supervised classification
  • Calculate accuracy assessment
02

Conditional Algorithm (SQL)

Uses ArcGIS Con tool with a SQL threshold condition to replace cloud pixels with cloud-free data.

  • Open cloud image + cloud-free image in ArcMap
  • Open Spatial Analyst → Conditional → Con tool
  • Apply SQL condition: value >= 243 (cloud reflectance threshold)
  • Output replaces high-reflectance cloud pixels
  • Apply supervised classification
  • Calculate accuracy assessment
03

ECDR Algorithm

Efficient Cloud Detection and Removal — uses multiple reference images to calculate a Cloud Replacement Pixel (CRP) value for each cloud pixel.

  • Detect clouds; load reference images RI1, RI2, RI3
  • Calculate D1 = RI1−RI2, D2 = RI2−RI3
  • Average: D = (D1 + D2) / 2
  • Calculate variance V between RI3 and D
  • CRP = RI3 ± V (based on sign of D1, D2)
  • Replace each cloud pixel with its CRP value
  • Apply supervised classification + accuracy assessment

Key Findings

Conclusion

The ECDR Algorithm consistently outperformed both the Brovey Image Fusion and Conditional Algorithm methods across all three study areas. By computing a Cloud Replacement Pixel value using variance between reference images, ECDR produced the most accurate reconstruction of cloud-affected pixels, resulting in significantly improved land cover classification accuracy and higher Kappa coefficients.

These results demonstrate that variance-based cloud pixel replacement is more effective than simple image fusion or threshold-based conditional replacement, particularly for complex urban and peri-urban landscapes.
